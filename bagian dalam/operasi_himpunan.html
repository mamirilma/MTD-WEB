Operasi Himpunan



Gabungan

Gabungan himpunan A dan B ditulis A ∪ B adalah himpunan semua objek yang menjadi anggota himpunan A atau himpunan B.

Contoh:
Diketahui: A = {1, 2, 3, 4, 5} B = {4, 5, 6, 7}
Tentukan A ∪ B
A ∪ B = {1, 2, 3, 4, 5, 6, 7}


Irisan

Irisan himpunan A dan B ditulis A ∩ B adalah himpunan semua objek yang menjadi anggota himpunan A sekaligus menjadi anggota himpunan B.

Notasi:
A ∩ B = { x | x ∈ A dan x ∈ B }

Contoh:
Misalkan A = {1, 2, 3} dan B = {3, 4, 5}
Maka A ∩ B = {3}


Selisih

Selisih himpunan A dan B, ditulis A − B, adalah himpunan semua objek yang menjadi anggota A tetapi bukan anggota B.

Notasi:
A − B = { x | x ∈ A dan x ∉ B }

Contoh:
Misalkan A = {1, 2, 3} dan B = {3, 4, 5}
Maka A − B = {1, 2}


Komplemen

Komplemen dari suatu himpunan A (ditulis A' atau Ac) adalah semua anggota himpunan semesta (S) yang tidak ada di dalam A.

Notasi:
A' atau Ac = { x | x ∈ S dan x ∉ A }

Contoh:
Misalkan S = {1, 2, 3, 4, 5} dan A = {2, 4}
Maka A' = {1, 3, 5}


Beda Setangkup (Beda Simetris)

Beda setangkup atau beda simetris antara A dan B ditulis A Δ B dan merupakan himpunan anggota yang berada di A atau di B tetapi bukan di keduanya sekaligus.

Notasi umum:
A Δ B = (A − B) ∪ (B − A)

Contoh:
Misalkan A = {1, 2, 3, 5} dan B = {2, 5, 6, 8}
A − B = {1, 3} dan B − A = {6, 8}
Maka A Δ B = {1, 3, 6, 8}
